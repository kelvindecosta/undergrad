# XML - Schema

## [`Contact`](Contact/Contact.xsd)

XML Schema for Contact details :

```
Contact
    - Name
    - Age
    - Country
    - Email
```

## [`PurchaseOrder`](PurchaseOrder/PurchaseOrder.xsd)

XML Schema for PurchaseOrder details :

```
PurchaseOrder
    - Item
        - Name
        - Qty
        - Price
    - ShippingAddress
        - Street
        - City
        - Pin
```
